# BNFとは
* 正式名称はBuckus-Naur form
* 拡張したものに [EBNF](./EBNF.md)がある
* BNFでは一つ一つの生成規則をA = a1a2という形式で表す。これはAをa1a2に展開できるという意味。
    * a1a2は0個以上の記号の列
    * それ以上展開出来ない記号とさらに展開される記号に分れる。
        * それ以上展開できない記号(終端記号)
        * さらに展開される記号(非終端記号)
            * 非終端記号は複数の生成規則にマッチしても良い。
    * このような [生成規則](./productionRule.md)で定義される文法を文脈自由文法と言う。
